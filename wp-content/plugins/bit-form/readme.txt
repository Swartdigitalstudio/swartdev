Here is the list of field types that you can use to create any type of WordPress form you want:

1. [Text Field](https://docs.form.bitapps.pro/wpbitform-textfield) - Text Field is to collect single line text from the user via the contact form. This is more like a generic field that can be used to collect name, email subject, product title, or any types of data that can be represented in a single line in your contact form.

2. [Multiline Field](https://docs.form.bitapps.pro/wpbitform-multilinetextfield) - Multiline Field is mainly used when you have to collect description or message from the user via the contact form. This field is also known as Textarea field.

3. [Email Field](https://docs.form.bitapps.pro/wpbitform-emailfield) - Collect email address from the user via the contact form. Email field has built-in validation to check if the email address is valid or not before submitting the form to protect your contact form from spam.

4. [Number Field](https://docs.form.bitapps.pro/wpbitform-numberfield) - Number field is very useful to collect numeric data such as age, phone number, zip code, etc. from the user via the contact form, survey form, feedback form etc.

5. [Select Field](https://docs.form.bitapps.pro/wpbitform-selectfield) - A native HTML select field to collect simple option based data from the user via the contact form, quiz form, calculator form, etc.

6. [Radio Button Field](https://docs.form.bitapps.pro/wpbitform-radiobuttonfield) - Radio button field collects single option based data as radio button from the user via the contact form, quiz form, survey form, feedback form, polls form etc.

7. [Checkbox Field](https://docs.form.bitapps.pro/wpbitform-checkboxfield) - Checkbox field collects multiple choice based data from the WordPress users via the contact form, quiz form, survey form, feedback form, booking form, poll form etc.

8. [Dropdown Field](https://docs.form.bitapps.pro/wpbitform-drop-downfield) - A customized dropdown field with the ability of multi-select to collect data from WP users via the contact form, quiz form, donation form, feedback form, etc.

9. [File Upload Field](https://docs.form.bitapps.pro/wpbitform-fileuploadfield) - A simple file upload field with multiple file upload option to collect files from the user via the contact form in WordPress.

10. [Advance File Upload Field](https://docs.form.bitapps.pro/wpbitform-advancedfileuploadfield) (Pro) - Drag and Drop File upload field with tremendous amount of features and customizations to collect files from the user via the feedback form, booking form, application form, complaint form, etc.
